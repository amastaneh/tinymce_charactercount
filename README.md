# TinyMCE Character Count
TinyMCE Character Count plugin adds character count functionality to TinyMCE.

### Initialization Example 1

```javascript
tinymce.init({
    plugins: "charactercount"
});
```

## Initialization Example 2

```javascript
tinyMCE.init({
        theme : "advanced",
        mode : "textareas",
        plugins : "charactercount"
});
```
